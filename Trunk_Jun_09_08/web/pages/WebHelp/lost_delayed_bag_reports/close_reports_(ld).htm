Close a Lost/Delayed Bag Report

 

 

Once a Lost/Delayed report has been resolved either by the bag(s) being found or a claim has been paid, the report should be closed.

 

To close a Lost/Delayed Bag report:

  1. Create and save a new Lost/Delayed Bag report.
     

  2. Select the Report Number link that is shown on the save confirmation screen.

    - or -

    Locate and view     an existing Lost/Delayed Bag report that is still listed as Open.
     

  3. From the section toolbar, click to display the Close Report screen.
     

  4. Enter the optional and required information in the appropriate fields for each bag.
     

  5. Click to save the information, assign a Lost/Delayed Report number and close the report.
     

Close Report Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Arrival Airline

Choose the airline the passenger arrived on from the drop-down list.

Arrival Flight

The flight number of the flight the passenger arrived on.

Arrival Date

Select the Date of arrival using the Date selector control.

Fault Airline

Choose the airline that is responsible for the Lost/Delayed bag. Once chosen, the appropriate Fault Stations will appear in a drop-down box.

Fault Station

Enter the Fault Station by pointing to the station code in the drop-down list and clicking on the selected fault station.

Reason for Loss

Enter the Reason for Loss by pointing to the description in the drop-down list and clicking on the selected reason for loss.

Remarks

Enter any comments or additional information about the missing bag. Each Remark may be a maximum of 1500 characters.  The characters still available box will count down from 1500 to 0 as text is entered into the field. You can also add or delete comments for each bag.
