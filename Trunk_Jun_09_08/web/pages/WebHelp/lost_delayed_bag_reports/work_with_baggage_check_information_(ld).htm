Work with Baggage Check Information

 

 

The Baggage Check Information section is for information that pertains to all the bags that were checked by the passenger(s) whether or not they are being reported as lost or delayed.

 

The claim check number for each mishandled bag must be added to the report.

 

To add general information about all bags to a Lost/Delayed Bag report:

  1. Create a new Lost/Delayed Bag report or locate and view an existing one.
     

  2. Click the button to jump to that section of the report.
     

  3. Enter the optional and required information in the appropriate fields as well as the claim check number of each bag.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To add an additional claim check to a Lost/Delayed Bag report:

  1. Locate and view an existing Lost/Delayed Bag report.
     

  2. Click the button to jump to that section of the report.
     

  3. Click to display an additional claim check number field.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To delete a claim check from a Lost/Delayed Bag report:

  1. Locate and view an existing Lost/Delayed Bag report.
     

  2. Click the button to jump to that section of the report.
     

  3. Click to remove the desired claim check.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

Baggage Check Information Fields

Field Name

Field Description

Claim Check Number

Enter the claim check number for each bag that has not been received by the passenger. If the claim check number is not known, enter the airline code of the carrier that originally checked the baggage.

Bags Checked Location

Select the location where the passenger checked their baggage by selecting the location from the dropdown list.  Select Unchecked if the baggage was not turned over to the airline for handling.

Number of Passengers

Enter the number of passengers that were traveling together and involved in the mishandled baggage report. The default is (1) passenger.

Number of Bags Checked

Enter the total number of bags checked by the passenger(s). The default is (1) bag checked.

Bags Received

Enter the total number of bags that were checked and received by the passenger(s) upon their arrival. The default is (0) bags.

Courtesy Report

Select Yes from the drop-down list if the Lost/Delayed Baggage report is being taken as a courtesy to the passenger.  The default is No.

TSA Inspection

Select Yes from the drop-down list if the passenger’s baggage was inspected by the Transportation Security Administration in their presence. The default is No.

Custom Cleared

Select Yes from the drop-down list if the passenger’s baggage cleared customs. The default is No.
