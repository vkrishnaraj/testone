Work with Express Lost/Delayed Bag Reports

 

 

A Lost/Delayed Bag Report is created when a passenger reports a bag missing when they arrive at their scheduled stopover or final destination. An Express Lost/Delayed Bag Report is a shortened version of the original form that only displays the required fields from each section. All other fields are set to their default value.

 

Regardless of which type of report is originally created, there is no difference between the two once the report is created.

 

To add a new Express Lost/Delayed Bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Express Lost/Delayed Bag from the Lost/Delayed menu.
     

  3. Enter the optional and required information in the appropriate fields for each section of the report.
     

  4. Click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
