Work with Lost/Delayed Bag Information

 

 

Baggage Information is recorded for each bag that a passenger is reporting as mishandled. Baggage Information contains descriptive information about each bag which is used in matching it with On-hand bags. It also records when the bag has been located and received by the station.

 

Fields within a Lost/Delayed Report can be modified by displaying the file and updating or making changes to the existing information. You also have the option to add or delete some sections in a file. A Delete button and Add button are displayed at the end of the section when this option is available. Click in a field to:

  • Add new information in a field that is currently blank
     

  • Change information by typing over existing data
     

  • Delete information by clearing the data in a field

Click on your browser back button to ignore all changes that have been made to a report that has not yet been saved. Or you can navigate to another Net-Tracer screen prior to saving the changes to ignore the updates.

 

To add an additional bag to a Lost/Delayed Bag report:

  1. Create a new Lost/Delayed Bag report or locate and view an existing one.
     

  2. Click the button to jump to the Baggage Information section of the report.
     

  3. Click to add an additional bag to the report.
     

  4. Enter the optional and required information in the appropriate fields.
     

  5. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.

Note

 

There is no limit to the number of damaged bags that can be added to a Damaged Bag report.

 

 

To edit an existing bag in a Lost/Delayed Bag report:

  1. Locate and view an existing Lost/Delayed Bag report.
     

  2. Click the button to jump to the Baggage Information section of the report.
     

  3. Make the desired changes to the damaged bag's information fields.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To delete a bag from a Lost/Delayed Bag report:

  1. Locate and view an existing Damaged Bag report.
     

  2. Click the button to jump to the Baggage Information section of the report.
     

  3. Click the button that corresponds with the bag that is to be deleted.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.

Important

 

A minimum of one Damage Bag Information field is required to complete a report.

 

 

Lost/Delayed Bag Information Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Name on Bag

 

Enter the name or initials on the bag if it is different than the passenger’s name.

Color

 

Select the color of the bag by pointing to the color in the drop-down list and clicking on the selected color.  Alternatively, the color can be selected by placing the cursor in the field and entering the first letter of the desired color. For example, repeatedly entering the letter B will in order select Black, Blue, Beige and then Brown.

Type

Select the baggage type by pointing to the type in the drop-down list and clicking on the selected type. Alternatively, the type can be selected by placing the cursor in the field and entering the first digit of the desired bag type. For example, repeatedly entering the number 2 will in order select 20 through 29. For bag types 1-9 select 0 as the beginning number.

Descriptive Elements

Three (3) drop down list are provided for entering External Descriptive Elements. The selections in each box are identical; W-wheels, S-straps, P-pockets, H-handle, C-combination lock, B-single item in a box and X for no external descriptive elements. These may be entered by pointing to the descriptive element in the drop-down list and clicking on the selected element or by placing the cursor in the field and entering the first letter of the descriptive element; W, S, P, H, C, B or X.

Manufacturer

 

Select the name of the luggage manufacturer by pointing to the name in the drop-down list and clicking on the selected manufacturer.

Contents

 

Place your cursor in the contents field and enter a description of the item and the damage for any content items that incurred damage.
