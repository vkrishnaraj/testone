Maintain Loss Codes

Loss codes are reasons used to establish proof of responsibility for and also to improve baggage mishandlings. A Loss Code and Fault Station are required to close a Lost/Delayed Baggage Report, Damage Report or Missing Article Report. Net-Tracer will prompt for assignment of both upon close. Reason for Loss Codes are customizable by airline. They may be 2-tiered, primary and secondary or single tiered, primary or secondary only.

 

The Loss Codes page lists the Reason for Loss Codes that are used by your airline.  Your airline may use standard IATA Reason for Loss Codes (see IATA Recommended Practice 1743E) or codes and descriptions that are customizable by airline.

 

The following tasks are associated with Maintaining Loss Codes:

  • Add a Loss Code

  • Edit a Loss Code

  • Delete a Loss Code
