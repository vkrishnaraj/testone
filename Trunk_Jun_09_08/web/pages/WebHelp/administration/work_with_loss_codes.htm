Work with Loss Codes

 

 

to edit an existing Loss Code:

  1. Locate and click the code number of the Loss Code you want to modify. This will display the Edit Loss Code screen.
     

  2. Enter the desired changes to the Description field.
     

  3. Click to accept the changes.

Note

 

When editing an existing Loss Code, all fields except for the Description field are ghosted (locked) and cannot be changed.

 

 

To delete a Loss Code:

  1. Check the boxes in the Delete Code column next to the desired Loss Code(s).
     

  2. Click the Batch Delete button.
     

  3. Click OK to confirm the deletion.
