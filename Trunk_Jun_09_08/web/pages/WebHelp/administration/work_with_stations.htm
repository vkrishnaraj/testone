Work with Stations

 

 

To edit an existing Station:

  1. Click on the Station's name in the Station Code column to open the Edit Station page. This will display the Station's current details.
     

  2. Make the desired changes to the Station fields.
     

  3. Click the Add Agents to Station button to add or delete Agents associated with the Station.
     

  4. Click to update the Station with the new information.

To delete a Station:

  1. Locate the desired Station's name in the Station Code column.
     

  2. Click the button on the row of the Station being removed.
     

  3. Click OK to confirm the deletion of the Station.
