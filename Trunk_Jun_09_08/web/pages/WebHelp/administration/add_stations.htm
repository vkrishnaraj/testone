Add Stations

 

 

To Add a new Station:

  1. Click the Add Station button at the bottom of the Maintain Stations screen. This will display the Edit Station screen.
     

  2. Enter the new Station's details.
     

  3. Click when you are finished or Add Agents to the Station.

To Add Agents to a Station:

  1. From the Edit Station screen, click the Add Agents to Station button.
     

  2. Check the boxes associated with the Agent's names. This will associate the Agents to the Station.
     

  3. Click the Add Agents to Station button to confirm the association of the selected Agents.
     

  4. Click to confirm the creation of the new Station.

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

 

Station Fields

Field Name

Field Description

Station Code *

Enter a valid three (3) letter station code for your airline.

Station Name *

Enter the official airport name for your station. The Station Name can be a maximum of 25 characters.

Language

The current language used in Net-Tracer is English. The Language value will be selected by choosing the language that is most commonly used by the agent in their country/region from the dropdown list. Click the down pointed arrow to display the list of available languages. Move your cursor over the selected language to highlight it and click to enter the data in the Language field.

Note

 

Future enhancements will allow for the support of multiple languages.

 

Company Code

Company code is a locked field. The code will default to the company code assigned in your user profile.

Street Address

Enter the physical address for the Baggage Service Office. The address can be a maximum of two (2) lines with twenty-five (25) characters on each line.

City

Enter the city name for the Baggage Service Office. The city can be a maximum of twenty-five (25) characters.

State

Select the state the Baggage Service Office is located in by highlighting the state name in drop-down list and clicking on the selected state.

Country

Select the country the Baggage Service Office is located in by highlighting the country name in drop-down list and clicking on the selected country.

Zip

Enter a valid 5 to 9 t postal code for the Baggage Service Office.

Phone

Enter the telephone number for the Baggage Service Office. The city can be a maximum of twenty-five (25) characters.

BSO Location

Enter brief description of the location of the Baggage Service Office in the airport. The BSO Location can be a maximum of sixty (60) characters.

Hours of Operation

Enter the hours of operation for the Baggage Service Office. The hours of operation can be a maximum of twenty-three (23) characters.
