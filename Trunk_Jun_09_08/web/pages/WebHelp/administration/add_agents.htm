Add Agents

 

 

To Add an new Agent:

  1. Log in to the Net-Tracer application.
     

  2. Select Maintain Agents from the Administration menu option on the main menu bar.
     

  3. Click the button at the bottom of the Maintain Agents screen. This will display the Edit Agent screen.
     

  4. Enter the user's profile information in the appropriate fields.
     

  5. Click to add the new Agent profile.
     

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

 

Agent Fields

Field Name

Field Description

Username *

Usernames are established by the Net-Tracer System Administrator at your airline. Enter 3-10 alpha/numeric characters. Usernames are not case sensitive. Special characters are not allowed.

Password *

 

A password consists of 4-10 alpha/numeric characters and is not case sensitive. Passwords are established by the designated airline’s System Administrator. If you have forgotten your password contact your System Administrator for further instructions.

Confirm Password *

Retype the password exactly as it was originally entered to confirm the password.

Company Code

Company code is a locked field. The code will default to the company code assigned in your user profile.

Station *

Select the Agent’s assigned station by pointing to the 3 letter code in the dropdown list and clicking on the selected station.  

Group *

Each user group in Net-Tracer is assigned a name and specific levels of access by your Net-Tracer System Administrator. Select the group you want to assign the Agent to by pointing and clicking on the Group name in the dropdown list.

First Name *

Type the Agent’s first name in the field. First names can be a maximum of 30 characters.

Middle Name

Type the Agent’s middle name in the field or double click in the field to select a name from the list of previously entered names. Middle names can be a maximum of 30 characters.

Last Name *

Type the Agent’s last name in the field. Last names can be a maximum of 30 characters.

Timeout *

Enter the number of minutes you want Net-Tracer to remain active when no recent entries have been made in the active session. Your session will expire after the specified amount of time entered in the time out field. The Timeout value can be a maximum of 3 numeric characters.

Important

 

Timeout values must be within the range of 30 to 200 minutes.

 

Active? *

This field indicates whether the user access to Net-Tracer is active or inactive. Select Yes from the dropdown list for active. Select No to indicate inactive. The default is Yes.

Default Timezone *

Select the time zone where the Agent is located from the Timezone dropdown list. Click the down pointed arrow to display the list of available time zones. Move your cursor over the selected time zone to highlight it and click to enter data in the Default Timezone field.

Current Timezone *

Select the time zone where the Agent is located from the Timezone dropdown list. Click the down pointed arrow to display the list of available time zones. Move your cursor over the selected time zone to highlight it, then click to enter the data in the Current Timezone field.

Default Language *

The default language used in Net-Tracer is English. The language value will be selected by choosing the language that is most commonly used by the agent in their country/region using the dropdown list. Click the down pointed arrow to display the list of available languages. Move your cursor over the selected language to highlight it and click to enter the data in the Default Language field.

Note

 

Future enhancements will allow for the support of multiple languages.

 

Current Language *

The current language used in Net-Tracer is English. The language value will be selected by choosing the language that is most commonly used by the agent in their country/region from the dropdown list. Click the down pointed arrow to display the list of available languages. Move your cursor over the selected language to highlight it and click to enter the data in the Current Language field.

Note

 

Future enhancements will allow for the support of multiple languages.

 

Currency *

Select the default currency that the Agent will use for file creations and claims payments using the dropdown list. Click the down pointed arrow to display the list of available currencies. Move your cursor over the selected currency to highlight it and click to enter the data in the Currency field.

Date Format *

Select the date format that the Agent will use for all date field entries in the Net-Tracer application from the dropdown list. Click the down pointed arrow to display the list of available date formats. Move your cursor over the selected date format to highlight it and click to enter the data in the Date Format field.

Time Format *

Select the time format that the Agent will use for all time field entries in the Net-Tracer application from the dropdown list. Click the down pointed arrow to display the list of available time formats. Move your cursor over the selected time format to highlight it and click to enter the data in the Time Format field.

Shift Description

Some airports assign their Agents to specific shifts. If applicable to your airport, select the shift that the Agent is assigned to using the dropdown list. Click the down pointed arrow to display the list of available shift descriptions. Move your cursor over the selected shift to highlight it and click to enter the data in the Shift Description field. The default value is None.
