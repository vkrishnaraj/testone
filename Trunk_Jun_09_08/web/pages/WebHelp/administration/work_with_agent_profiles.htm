Work with Agent Profiles

 

 

To edit an existing Agent Profile:

  1. Click on the Agent's name in the Username column to open the Edit Agent page. This will display the Agent’s current profile.
     

  2. Make the desired changes to the profile fields.
     

  3. Click to update the profile with the new information.

To delete an Agent Profile:

  1. Locate the desired Agent's name in the Username column.
     

  2. Click the button on the row of the Agent being removed.
     

  3. Click OK to confirm the deletion of the Agent's Profile.

To maintain or modify your User Profile:

The Maintain User Profile menu option allows you to modify certain details of your own user profile. For example, you can change your password.

  1. Select Maintain User Profile from the Administration menu.
     

  2. Make the desired changes to the profile fields.
     

  3. Click to update the profile with the new information.

Note

 

Some fields require Administrator permissions to change.
