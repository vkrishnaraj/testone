Administration

 

 

Administration functionality allows an airline to manage system settings, work flow processes, contact information and user accounts.  Each airline has an assigned Net-Tracer Administrator.  The Administrator has access to change system settings and add or delete a User.  Each user will have access to maintain their Net-Tracer profile.

 

The following options are available by selecting the Administrator option from the main menu bar:

  • Maintain User Profile

  • Maintain Groups

  • Maintain Stations

  • Maintain Agents

  • Maintain Loss Codes

  • Maintain Shifts
