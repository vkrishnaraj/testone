Work with Missing Articles Reports

 

 

A Missing Articles Report is created when a passenger alleges that their baggage has been pilfered. Missing Articles Reports are created when a passenger reports that their baggage is missing contents. Items missing due to exterior damage of the bag may be included in the Damage Bag Report.

 

To add a new Missing Articles report:

  1. Log in to the Net-Tracer application.
     

  2. Select Add Missing Articles from the Missing Articles menu.
     

  3. Enter the optional and required information in the appropriate fields for each section of the report.
     

  4. Click to submit the Missing Articles report.
     

To locate, view or edit an existing Missing Articles report:

  1. Log in to the Net-Tracer application.
     

  2. Select Search Report(s) from the Search menu on the main menu bar.
     

  3. Enter any known information about the report in the appropriate fields.
     

  4. Click to locate the Missing Articles report.
     

  5. Once the Results section is displayed, select the link shown in the Report Number column for the report you wish to view.
     

  6. Make any desired changes.
     

  7. Click to save the changes and update the report in the system.
     

Report Information Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Report Number

Net-Tracer Report Numbers consist of the 3 letter station code, the 2 letter airline code and a minimum of 7 numeric characters. The Report Number will automatically be assigned when the record is saved.

Report Status

 

Report Status will display as Temporary until the record is saved.  

Agent Initials

Agent Initials is a locked field. The initials will default to the agent initials assigned in your user profile.

Station Created

Station Created is a locked field. The station code will default to the station assigned in your user profile.

Station Assigned

In most cases Station Assigned will be the same as the Station Created. However in cases where the passenger has continued his travel to another city the file may be reassigned to the new station.

Report Date/Time

The Date and Time is computer generated based on the date and time at the station where the report is created.

Record Locator

Enter the passenger’s airline reservation record locator.

Ticket Number

Enter the full airline ticket number. If the full ticket number is not available enter any portion of the number that is known.

Non-Revenue

Select Yes if the passenger is traveling non-revenue, space available or No if the passenger is traveling on a revenue ticket.  The default is No.

Report Method

Select a method of reporting from the drop-down menu.
