Help Viewer Overview

 

 

The Help viewer is the window you are looking at now. It is used to provide Help that is specific to certain NetTracer features, such as reports, tasks, and administration functions. You can adjust the size of the Help viewer or hide the navigation pane if you need to place the Help viewer beside the program you are using.

 

For immediate help when using NetTracer, you can display context-sensitive Help about a particular item or section. Click next to the title of the section or feature you need help on.
