Work with Incoming Bags

 

 

The Incoming Bags screen displays a list of found bags that are being forwarded to your station for handling.

 

To view the details about an incoming On-hand Bag:

  1. Log in to the Net-Tracer application.
     

  2. Select Incoming Bags from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Incoming Bags link on the Task Manager Home screen.
     

  3. Select the associated On-hand Bag number link in the On-hand Bag column to view the desired bag's details.
     

To view the original message details forwarded with the On-hand Bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Incoming Bags from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Incoming Bags link on the Task Manager Home screen.
     

  3. Select the associated Details link in the Forward column to view the forwarded message details.
     

To receive and On-hand Bag at your station:

  1. Log in to the Net-Tracer application.
     

  2. Select Incoming Bags from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Incoming Bags link on the Task Manager Home screen.
     

  3. Select the associated Receive link in the Action column to accept or receive the bag at your station.

Note

 

Receiving an On-hand Bag will change the Holding station designation in the On-hand report to your station and close the report. It will also remove the On-hand bag report number from the Incoming Bag list.
