Task Manager Summary

 

 

The Task Manager Home Page is the log in default page.

 

The following options are available from the Task Manager Home page or by selecting the Task Manager option from the main menu bar:

  • Task Manager Home

  • View Trace Results

  • Inbox

  • Forward Bags to LZ

  • Claims To Be Processed

  • Incoming Bags

  • View Incoming Requests

  • Other Tasks

  • View Created Requests

  • Temporary Report(s)

  • Temporary On-hand(s)

  • Mass On-hand(s)
