Other Tasks

 

 

Other Tasks is a list of scheduled activities assigned to a station or a specific file. Activities are assigned by fellow agents with a priority of high, medium or low. The activity will also be assigned a reminder date, if applicable, and a due date for completion. Click on the Other Tasks option to open the Search Task page.

 

Search Tasks let’s you quickly search through your list of tasks to locate specific tasks.

 

To search for a specific task:

  1. Log in to the Net-Tracer application.
     

  2. Select Other Tasks from the Task Manager menu.
     

  3. Enter the search criteria using the fields provided in the Search Tasks section.
     

  4. Click to locate the desired task(s).
     

  5. Click to clear the fields and start a new search.
     

To view or edit an existing task:

  1. Locate or search for the desired task in the Tasks list.
     

  2. Click the Edit link shown in the Edit/View column associated with the task.
     

  3. View the details or make the necessary changes.
     

  4. Click to confirm the changes or to close the task without any changes.
     

To create a new task:

  1. Locate or search for the desired task in the Tasks list.
     

  2. Click at the bottom of the screen to display the Tasks entry form.
     

  3. Enter the task's required and optional details like assigning an Agent, a due date, a priority and a status.
     

  4. Associate a report, if necessary, and enter any notes or remarks.
     

  5. Click to confirm the task's creation.

Note

 

Remarks may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.

 

 

To mark a task as completed:

  1. Locate or search for the desired task in the Tasks list.
     

  2. Click the Edit link shown in the Edit/View column associated with the task.
     

  3. Change the designation shown in the Status dropdown box to Completed.
     

  4. Click to confirm the changes.

Note

 

A task's status can be Not Started, Pending, In Progress, Deferred, Completed or Not Completed.

 

 

Other Task Search Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Status

You have the option to search for tasks by ALL statuses or by a specific task status (Not Started, Pending, In Progress, Deferred and Completed). Select the Status by pointing to the status in the drop-down list and clicking on the selected status. Add additional search criteria or click on the Search button to search by Status only.

Date Range

You have the option to search for tasks by a date or date range. Net-Tracer will search tasks with a Due Date or Reminder Date that match the dates entered. Add additional search criteria or click on the Search button to search by Date Range only.

Report Number

You have the option to search for tasks by the Report Number. Net-Tracer will search all file types with matching report numbers that have tasks assigned. Add additional search criteria or click on the Search button to search by Report Number only.

Task Form Fields

Field Name

Field Description

Report Number

Enter a valid Net-Tracer report number.  If the task is not associated with a specific report the field should be left blank.

Type

Select the type by pointing to the appropriate report type in the drop-down list and clicking on the selected type.  The default is Other which indicates that the task is not associated with a specific report.

Created By

Created By is a locked field.  The entry will default to the Username for log-in.

Station

Station is a locked field.  The station code will default to the station assigned in your user profile.

Subject*

Enter a subject name or brief description for the task.

Due Date*

Enter the date the task is due, if applicable.  Date must be entered in the format assigned in your user profile.

Due Time

Enter the time the task is due, if applicable.  Time must be entered in the format assigned in your user profile.

Reminder Date

Enter the date the reminder for the task is due, if applicable.  Date must be entered in the format assigned in your user profile.

Reminder Time

Enter the time the reminder for the task is due, if applicable.  Time must be entered in the format assigned in your user profile.

Priority

Select the Priority for the task by pointing to the appropriate priority level in the drop-down list and clicking on the selected priority.  The default is Medium.

Status

Select the Status for the task by pointing to the appropriate status type in the drop-down list and clicking on the selected status.  The default is Not Started.

Remarks

Remark may be a maximum of 255 characters.  The characters still available box will count down from 255 to 0 as the Remark is entered.
