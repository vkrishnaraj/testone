View Created Requests

 

 

The View Created Requests screen displays a list of requests for found baggage that have:

  • Been made by your station for On-hand bags held by another station or airline and
     

  • Match Lost/Delayed Baggage Reports that are under your station's control

The Search Request On-hand section let’s you quickly look through your list of requests for On-hand bags for a specific one. The Status column shown in the Request On-hand list indicates if the request for the On-hand bag is still Open (no action has been taken), the bag has been Forwarded, the request has been Denied or the request has been Closed.

 

To search for an On-hand bag request:

  1. Log in to the Net-Tracer application.
     

  2. Select View Created Requests from the Task Manager menu.

    - or -

    Select the View Created Requests link on the Task Manager Home screen.
     

  3. Enter the search criteria using the fields provided in the Search Request On-hand section.
     

  4. Click to locate the desired request(s).
     

  5. Click to clear the fields and start a new search.
     

To view any of the details for a Created Request:

  1. Locate or search for the desired request in the Request On-hand list.
     

  2. Click the link shown in the On-hand Bag column to display the associated On-hand bag report.

    - or -

    Click the link shown in the Report Number column to display the associated Lost/Delayed report.

    - or -

    Click the Details link shown in the Match column to display the specifics if the request is due to a match.

    - or -

    Click the Details link shown in the Request column to display the details of the request for the On-hand bag.
     

Search Request On-hand Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Status

You have the option to search for requests for On-hand bags by ALL report statuses or by a specific status (Open, Closed, Forwarded, Denied).  Select the Status by pointing to the status in the drop-down list and clicking on the selected status.  Add additional search criteria or click on the Search button to search by Status only.

On-hand Report Number

You have the option to search for requests for On-hand bags by the On-hand Report Number.  Net-Tracer will search all On-hand Reports for matching report numbers.  Add additional search criteria or click on the Search button to search by On-hand Report Number only.

Report Number

You have the option to search for requests for On-hand bags by Report Number.  Net-Tracer will search all report types for matching report numbers.  Add additional search criteria or click on the Search button to search by Report Number only.
