Inbox

 

 

You can use Net-Tracer Mail to send and receive emails from stations within your company or other airlines. The Inbox is a public email folder assigned to the station. All users assigned to the station will have permission to access the Inbox, view and reply to messages and create or delete messages.

 

The Inbox displays a list of messages received by the station. The number of rows per page option will appear after 15 messages have accrued to the Inbox. Messages are sorted by date and time received with the most recent message displayed first. The Status column indicates whether the message is “New,” has already been “Seen” or someone has already “Replied” to the message.

 

The Search Inbox feature let’s you quickly search through your list of emails to locate specific messages.

 

To locate a specific message in the Inbox:

  1. Log in to the Net-Tracer application.
     

  2. Select Inbox from the Task Manager menu.

    - or -

    Select the Inbox link on the Task Manager Home screen.
     

  3. Enter the search criteria using the fields provided in the Search Inbox section.
     

  4. Click to locate the desired message(s).
     

  5. Click to clear the fields and start a new search.
     

To open a message in the Inbox:

  1. Log in to the Net-Tracer application.
     

  2. Select Inbox from the Task Manager menu.

    - or -

    Select the Inbox link on the Task Manager Home screen.
     

  3. Click on the link listed in the subject column for the message you want to view.
     

To send a new message:

  1. Log in to the Net-Tracer application.
     

  2. Select Inbox from the Task Manager menu.

    - or -

    Select the Inbox link on the Task Manager Home screen.
     

  3. Click to view the Create Message screen.
     

  4. Enter a Report Number, if the message pertains to a specific file and select the file type from the dropdown list.
     

  5. Select the recipient airline station code from the To*: dropdown.
     

  6. If more than one recipient is desired, click to display an additional dropdown list.
     

  7. Enter the subject and type your message in the fields provided.
     

  8. Click on the to complete and send the message.

Note

 

Each Message may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.

 

 

To Reply to a message in the Inbox:

  1. Open the message that you want to reply to from the Inbox.
     

  2. From the Message window, click to display the Create Message screen.
     

  3. Enter a Report Number, if the message pertains to a specific file and select the file type from the dropdown list.
     

  4. If additional recipients other than the original sender(s) are desired, click to display an additional dropdown list.
     

  5. Enter the subject and type your message in the fields provided.
     

  6. Click on the to complete and send the message.

Note

 

The To*: field will already be filled in with the station code(s) of the original sender(s).

 

Note

 

Each Message may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.

 

 

To delete a message from the Inbox:

  1. Open the message that you want to delete from the Inbox.
     

  2. From the Message window, click to remove message.

Note

 

The text Message has been deleted will be displayed at the top of the Inbox screen confirming the message deletion.
