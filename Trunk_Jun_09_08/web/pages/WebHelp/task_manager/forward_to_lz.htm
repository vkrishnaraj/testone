Forward Bags to LZ

 

 

The Forward to LZ task displays a list of On-hand report numbers that apply to bags found at your station that need to be sent to LZ for further tracing. After a specified number of days determined by your company, the system will mark an On-hand for forwarding to LZ.

 

To view the details about an On-hand Bag:

  1. Log in to the Net-Tracer application.
     

  2. Select Forward Bags to LZ from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Forward Bags to LZ link on the Task Manager Home screen.
     

  3. Select the associated On-hand Bag number link in the On-hand Report Number column to view the desired bag's details.

To forward a single bag to LZ:

  1. Log in to the Net-Tracer application.
     

  2. Select Forward Bags to LZ from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Forward Bags to LZ link on the Task Manager Home screen
     

  3. Select the Forward link in the Forward On-hand column associated with the desired bag.
     

  4. Enter the Destination Station and an Expedite Tag Number.
     

  5. Add any known Baggage Itineraries for the bag being forwarded.
     

  6. Finally enter any instructions or notes in the Message field and click to complete the process.

Important

 

The origination (From) and destination (To) city, airline code, flight number and departure date are required for each segment. Optional fields include the arrival date of the flight, Scheduled Departure Time, Scheduled Arrival Time, Actual Departure Time and Actual Arrival Time.

 

Note

 

Each Message may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.

To forward multiple bags to the LZ:

  1. Log in to the Net-Tracer application.
     

  2. Select Forward Bags to LZ from the Task Manager menu on the Net-Tracer main menu bar.

    - or -

    Select the Forward Bags to LZ link on the Task Manager Home screen.
     

  3. Identify all of the bags to be forwarded by selecting their associated checkbox () in the Batch Forward column.
     

  4. Click Batch to display the Forward On-hand Bag screen.
     

  5. Enter an Expedite Tag Number for each bag and then choose the Destination Station from the dropdown box.
     

  6. Add any known Baggage Itineraries for the bag(s) being forwarded.
     

  7. Finally enter any instructions or notes in the Message field and click to complete the process.

Important

 

The origination (From) and destination (To) city, airline code, flight number and departure date are required for each segment. Optional fields include the arrival date of the flight, Scheduled Departure Time, Scheduled Arrival Time, Actual Departure Time and Actual Arrival Time.

 

Note

 

Each Message may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.
