View Incoming Requests

 

 

The View Incoming Request screen displays a list of requests that have been made by other stations or airlines for On-hand bags that are being held at your station.

 

Click on the On-hand Bag report number to display the On-hand bag report. Click on the Report Number to display the passenger file.  If the request for the bag is due to a match, clicking “Details” in the Match column will display the match details.  Clicking on “Details” in the Request column will display the details of the request for the On-hand bag.

 

To view any of the details for an Incoming Request:

  1. Log in to the Net-Tracer application.
     

    Select View Incoming Requests from the Task Manager menu.

    - or -

    Select the View Incoming Requests link on the Task Manager Home screen.
     

  2. Locate the desired request in the Incoming Request On-hand list.
     

  3. Click the link shown in the On-hand Bag column to display the associated On-hand bag report.

    - or -

    Click the link shown in the Report Number column to display the associated Lost/Delayed report.

    - or -

    Click the Details link shown in the Match column to display the specifics if the request is due to a match.

    - or -

    Click the Details link shown in the Request column to display the details of the request for the On-hand bag.
     

To deny an incoming request that is for the wrong On-hand bag:

  1. Log in to the Net-Tracer application.
     

    Select View Incoming Requests from the Task Manager menu.

    - or -

    Select the View Incoming Requests link on the Task Manager Home screen.

     

  2. Locate the desired request in the Incoming Request On-hand list.
     

  3. Click the Deny link shown in the Deny column if the request is for the wrong On-hand bag.
     

  4. Enter a reason for denial in the appropriate field.
     

  5. Click to complete the process and deny the request.

Note

 

When denying a request, the Deny Request On-hand form will display the link to the On-hand Bag Report and the Report Number. The Requesting Station will default to the airline and station that requested the bag and the Status will show Open. The Reason for Request field will show the message from the station requesting the On-hand bag.

 

Note

 

Remarks may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Reason for Denial field.

 

 

To forward a bag to the requesting station:

  1. Log in to the Net-Tracer application.
     

    Select View Incoming Requests from the Task Manager menu.

    - or -

    Select the View Incoming Requests link on the Task Manager Home screen.

     

  2. Locate the desired request in the Incoming Request On-hand list.
     

  3. Click the Forward link shown in the Forward column to send the bag to the requesting station.
     

  4. Confirm the Destination Station and enter an Expedite Tag Number.
     

  5. Add the Baggage Itinerary for routing the bag to the requesting station.
     

  6. Finally enter any instructions or notes in the Message field and click to change the status to In Transit.

Note

 

Remarks may be a maximum of 1500 characters. The characters still available box will count down from 1500 to 0 as text is entered in the Message field.
