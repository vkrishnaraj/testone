View Trace Results

 

 

The View Trace Results screen displays a list of all Net-Tracer system matches for On-hand bags matching a Lost/Delayed Baggage Report that is under your station’s control.

 

Passive tracing has 2 categories in Task Manager for match message delivery. Messages will be color coded within each category based on the percentage of the match:

  • Category 1 matches on primary items (claim check, frequent flyer ID, email, or record locator) and are 85% or higher.
     

  • Category 2 includes matches between 20% - 85%. However if none of the primary items match, then the minimum percentage will be 60%.

Net-Tracer will automatically stop tracing on a specified Claim Check and corresponding On-hand ID in a Lost/Delayed Report when a match has been accepted via Match Details. The Lost/Delayed Report will be updated with the matching On-hand Report number and a hyperlink to the file.

 

Tip

 

The number of rows per page option appears at the bottom of the screen. The number of match messages to view per page can be adjusted by entering the number of rows (1-99) desired per page and clicking the button.

 

 

To sort or filter matches for Lost/Delayed Baggage reports by match status:

  1. Log in to the Net-Tracer application.
     

  2. Select View Trace Results from the Task Manager menu.

    - or -

    Select the View Trace Results link on the Task Manager Home screen.
     

  3. Select the sort criteria by clicking the check box () in front of each desired option.
     

  4. Click to sort the list based on the new Criteria.

Note

 

The available match status options are Open, Matched, Rejected and Closed.

 

Tip

 

You can use the % character for wildcard searches. Examples for using wildcards to search by Report Number are:

§  A% to retrieve reports with city codes beginning with the letter A

§  %2% to retrieve reports with the number 2 anywhere in the report number

§  %2 to retrieve reports with report numbers ending with the number 2

 

 

To view any of the details for a trace result listing:

  1. Sort the Trace Results list by the Open status.

     

  2. Locate the desired result in the Trace Results list.
     

  3. Click the link shown in the On-hand Bag column to display the associated On-hand bag report.

    - or -

    Click the link shown in the Report Number column to display the associated Lost/Delayed report.

    - or -

    Click the Details link shown in the Match column to display the specifics if the request is due to a match.

    - or -

    Click the Details link shown in the Request column to display the details of the request for the On-hand bag.

Note

 

The percentage column will show the probability percentage of the On-hand Report match to the Lost/Delayed Baggage Report based on Net-Tracer matching parameters.

 

 

To confirm a match based on a trace result:

  1. Sort the Trace Results list by the Open status.

     

  2. Locate the desired result in the Trace Results list.
     

  3. Click the Details link shown in the Details column to display the View Trace Result Details screen.
     

  4. Click to display the match options.
     

  5. Choose a claim check for the bag match or a bag for the claim check match.
     

  6. Click again to verify and complete the match process.
     

To create a request for an On-hand bag:

  1. Sort the Trace Results list by the Open status.

     

  2. Locate the desired result in the Trace Results list.
     

  3. Click the Initiate Request On-hand link shown in the Action column to create a transfer request for an On-hand bag.
     

  4. When the Request On-hand Baggage screen displays, enter a reason for the request.
     

  5. Click to submit the transfer request.
     

To reject a match that is invalid:

  1. Sort the Trace Results list by the Open status.

     

  2. Locate the desired result in the Trace Results list.
     

  3. Review the details of the match.
     

  4. Click the Reject link shown in the Action column to reject the trace result.
     

To undo a rejected match:

  1. Sort the Trace Results list by the Rejected status.

     

  2. Locate the desired result in the Trace Results list.
     

  3. Click the Undo Reject link shown in the Action column to reopen the match status.
     

Trace Results Filter Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Report Number

Enter a valid Net-Tracer report number.  If the task is not associated with a specific report the field should be left blank.

On-hand Report Number

You have the option to search for requests for On-hand bags by the On-hand Report Number.  Net-Tracer will search all On-hand Reports for matching report numbers.  Add additional search criteria or click on the Search button to search by On-hand Report Number only.

Match Status

Open, Confirmed, Rejected and/or Closed check boxes.
