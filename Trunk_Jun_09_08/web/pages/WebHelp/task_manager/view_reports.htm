View Report(s)

 

 

The View Temporary Report(s) task option displays a list of all Lost/Delayed Reports that were saved as temporary. The View Temporary On-hand(s) task option displays a list of all On-hand Reports saved as temporary. Temporary reports have not been submitted for tracing.

 

View Mass On-hands lists all reports that were created using the Mass On-hand option.

 

To view a list of temporary reports:

  1. Log in to the Net-Tracer application.
     

  2. Select View Temporary Report(s) from the Task Manager menu.

    - or -

    Select the View Temporary Report(s) link on the Task Manager Home screen.
     

To view a list of temporary on-hands:

  1. Log in to the Net-Tracer application.
     

  2. Select View Temporary On-hand(s) from the Task Manager menu.

    - or -

    Select the View Temporary On-hand(s) link on the Task Manager Home screen.
     

To view a list of mass on-hands:

  1. Log in to the Net-Tracer application.
     

  2. Select View Temporary On-hand(s) from the Task Manager menu.

    - or -

    Select the View Temporary On-hand(s) link on the Task Manager Home screen.
     

To view any report's details:

  1. View Temporary Report(s), Temporary On-hand(s) or Mass On-hand(s).
     

  2. Locate the desired report in the list and click on the report number link.
