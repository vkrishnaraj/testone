Task Manager Home

 

 

Task Manager allows a user to view a list of activities assigned to a station or associated with a report as it moves through the work flow process. You can use Net-Tracer™ Inbox to send messages to other stations or airlines.

 

The Task Manager list displays the title of each subfolder and the number of items contained in the folder.

 

To view a specific subfolder:

  1. Click on the title of the subfolder listed in the Tasks column.
