Creating Recovery Reports

 

 

To create a Recovery report:

  1. Log in to the Net-Tracer application.
     

  2. Select Recovery from the Analytical Reports option on the main menu bar.
     

  3. Enter the required data in the appropriate field.
     

  4. Click Create Report to generate a report using the selected data.

Note

 

Once the new report is generated, you will see a new link appear at the bottom of the screen.

  1. Select the Click Here To View The Report link. The report will be displayed in PDF format.
     

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

 

Recovery Report Fields

Field Name

Field Description

Report Type

You have the option to select ALL or a specific report type (Lost/Delayed, Damaged or Missing Articles) from the drop-down list.

Date Range *

 

A date range is required and may be requested for a period of up to 12 months prior to the current date.  Each airline has the option to restrict the date range to a period of less than 12 months; however, it can not be extended beyond 12 months.

Assigned Station

The report may be generated for ALL stations or for a specific station.
