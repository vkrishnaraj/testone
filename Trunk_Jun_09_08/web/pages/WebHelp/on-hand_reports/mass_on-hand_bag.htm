Mass On-hand Bag

 

 

This report is used to enter 1 to 40 On-hand bag tag numbers with one entry.

 

The Mass On-hand entry allows you to input or scan up to 40 bag tag numbers in a single entry. Mass on-hand files are stored in Net-Tracer for 24 hours. After 24 hours Net-Tracer will automatically send these On-hand Report numbers to a Task Manager file where station agents will be instructed to create a complete On-hand Report or the temporary Mass On-hand Report will be deleted from the system.

 

To add a new Mass On-hand bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Mass On-hand Bag from the On-hand menu.
     

  3. Enter up to 40 Bag Tag Numbers, one in each field.
     

  4. Click to save the report and begin to match the new record with all open On-hand bags.
