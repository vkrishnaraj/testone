Work with Remarks

 

 

Each Remark may be a maximum of 1500 characters.  The characters still available box will count down from 1500 to 0 as text is entered into the field.

 

To add an additional Remark to an On-hand Bag report:

  1. Create a new On-hand Bag report or locate and view an existing one.
     

  2. Click the navigation button to jump to the Remarks section of the report.
     

  3. Click and enter any desired comments into the newly added text box.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To edit an existing Remark in an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the navigation button to jump to the Remarks section of the report.
     

  3. Make the desired changes to the remark field.
     

  4. Scroll to the end of the report and click to accept the changes.
     

To delete a Remark from an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the navigation button to jump to the Remarks section of the report.
     

  3. Click to delete the associated Remarks field.
     

  4. Scroll to the end of the report and click to accept the changes.

Note

 

There is no limit to the number of remarks that can be added to an On-hand Bag report.
