Work with Baggage Inventory

 

 

The Baggage Inventory section contains descriptive information about the contents of each bag on-hand.

 

Fields within an On-hand Bag Report can be modified by displaying the file and updating or making changes to the existing information. You also have the option to add or delete some sections in a file. A Delete button and Add button are displayed at the end of the section when this option is available. Click in a field to:

  • Add new information in a field that is currently blank
     

  • Change information by typing over existing data
     

  • Delete information by clearing the data in a field

Click on your browser back button to ignore all changes that have been made to a report that has not yet been saved. Or you can navigate to another Net-Tracer screen prior to saving the changes to ignore the updates.

 

To add additional content to an On-hand Bag report:

  1. Create a new On-hand Bag report or locate and view an existing one.
     

  2. Click the button to jump to the Baggage Inventory section of the report.
     

  3. Click to add additional content to the report.
     

  4. Enter the optional and required information in the appropriate fields.
     

  5. Attach digital photos to the report, if available.
     

  6. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.

Note

 

There is no limit to the content that can be added to an On-hand Bag report.

 

 

To edit existing content in an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the button to jump to the Baggage Inventory section of the report.
     

  3. Make the desired changes to the damaged bag's information fields.
     

  4. Scroll to the end of the report and click to accept the changes.
     

To delete content from an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the button to jump to the Baggage Inventory section of the report.
     

  3. Click the button that corresponds with the content that is to be deleted.
     

  4. Scroll to the end of the report and click to accept the changes.

Important

 

A minimum of one Damage Bag Information field is required to complete a report.

 

 

To upload a digital photo of a bag to an On-hand Bag report:

  1. Create a new On-hand Bag report or locate and view an existing one.
     

  2. Click the button to jump to the Photos section of the report.
     

  3. Click to upload a photo.
     

  4. Choose the file name of the photo you want to add and click .
     

  5. Click to upload the digital photo to the report.
     

  6. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.

Note

 

Repeat steps 3 through 5 to add additional photos.

 

Tip

 

Click on the thumbnail of the photo to enlarge the image.

 

 

To delete a digital photo from an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the button to jump to the Photos section of the report.
     

  3. Click below the desired image to remove the photo from the report.
     

  4. Scroll to the end of the report and click to accept the changes.
