Work with Passenger Information

 

 

The Passenger Information section allows you to enter contact information about the primary passenger as well as contact information about any additional passengers.

 

To add an additional passenger or address to an On-hand Bag report:

  1. Create a new On-hand Bag report or locate and view an existing one.
     

  2. Click the button to jump to the Passenger Information section of the report.
     

  3. Click to add an Additional Passenger section to the report.
     

  4. Enter the required and any optional information known.
     

  5. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To edit an existing passenger or address in an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the button to jump to the Passenger Information section of the report.
     

  3. Make the desired changes to the passenger information fields.
     

  4. Scroll to the end of the report and click to accept the changes.
     

To delete a passenger or address from an On-hand Bag report:

  1. Locate and view an existing On-hand Bag report.
     

  2. Click the button to jump to the Passenger Information section of the report.
     

  3. Click to remove the desired passenger or address from the report.
     

  4. Scroll to the end of the report and click to accept the changes.
     

Passenger Information Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Passenger’s Name

Enter the passenger’s last name, first name and middle initial. Partial information may be entered if the full name is not available. Middle initial is not required.

Salutation

Select the appropriate salutation from the drop down box.

Job Title

Enter the passenger’s job title if known.

Driver’s License

These two fields are used primarily by an airline’s baggage claims officer. If applicable select the state from the drop-down box. Tab to the next box and enter the full driver's license number.

Common Number (Passport)

Enter the passenger’s passport number or other government issued document number.

Country of Issue

Select the name of the country where the passport or other government document was issued from the dropdown menu.

Airline Membership

If the passenger is a member of an airline frequent traveler program, select the applicable code for that airline from the drop-down menu.

Membership Number

Enter the passenger’s frequent traveler membership number.  

Membership Status

Enter the program status level of the member.

Address 1

This section allows you to enter all known primary address information for the passenger including street, city, state/province, zip and country.

Address 2

This section allows you to enter a second address for the passenger including street, city, state/province, zip and country.

Telephone

Enter all known telephone numbers for the passenger. Options include Home, Business, Mobile, Pager and Fax.

E-mail

Enter the passenger’s full e-mail address.

Hotel

If the passenger is staying at a local hotel enter the name of the hotel and location. Otherwise enter the address and telephone number of the location where the bag is to be delivered.
