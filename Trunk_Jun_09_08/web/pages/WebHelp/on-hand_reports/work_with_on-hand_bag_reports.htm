Work with On-hand Bags

 

 

An On-hand Baggage Report should be completed immediately for each Lost/Delayed bag so that Net-Tracer can begin the matching process.

 

On-hand Bag reports should be created as soon as possible after an unclaimed bag has been received at your station or for any untagged baggage. System tracing will immediately begin matching the On-hand Bag Report to all open Lost/Delayed Reports upon entry into the Net-Tracer system.

 

To add a new On-hand bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Add On-hand Bag from the On-hand menu.
     

  3. Enter the optional and required information in the appropriate fields for each section of the report.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

To locate or view an existing On-hand Bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Search Report(s) from the Search menu on the main menu bar.
     

  3. Enter any known information about the report in the appropriate fields.
     

  4. Click to locate the On-hand Bag report.
     

  5. Once the Results section is displayed, select the link shown in the Report Number column for the report you wish to view.
     

Baggage Information Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

On-hand Report Number

Net-Tracer Report Numbers consist of the 3 letter station code, the 2 letter airline code and a minimum of 7 numeric characters. The Report Number will automatically be assigned when the record is saved.

Report Status

Report Status will display as New until the record is saved.

Agent Initials

Agent Initials is a locked field. The initials will default to the agent initials assigned in your user profile.

Date/Time Created

The Date and Time is computer generated based on the date and time at the station where the report is created.

Station Found At

Station Found At is a locked field. The station code will default to the station assigned in your user profile.

Holding Station

In most cases Holding Station will be the same as the Station Found At.  However in cases where the passenger has continued his travel to another city the file may be reassigned to the new station.  Holding Stations may not be reassigned until after the On-hand Baggage file has been created.

Name on Bag

 

Enter the name or initials on the bag if it is different than the passenger’s name.

Bag Tag Number

Enter the complete bag tag number. If the full bag tag number is not available enter any portion of the tag that is known. Net-Tracer will trace based on the string of characters available and provide potential matches. Leave the field blank if the bag tag number is unknown.

Storage Location

Enter the storage location in the BSO or the LZ office of the On-hand bag, if applicable. The maximum number of characters for the storage location is 50 characters.

Airline Membership

If the passenger is a member of an airline frequent traveler program, select the applicable code for that airline from the drop-down menu.

Membership Number

Enter the passenger’s frequent traveler membership number.

Record Locator

Enter the passenger’s airline reservation record locator.

Color

 

Select the color of the bag by pointing to the color in the drop-down list and clicking on the selected color.  Alternatively, the color can be selected by placing the cursor in the field and entering the first letter of the desired color. For example, repeatedly entering the letter B will in order select Black, Blue, Beige and then Brown.

Type

Select the baggage type by pointing to the type in the drop-down list and clicking on the selected type. Alternatively, the type can be selected by placing the cursor in the field and entering the first digit of the desired bag type. For example, repeatedly entering the number 2 will in order select 20 through 29. For bag types 1-9 select 0 as the beginning number.

Descriptive Elements

Three (3) drop down list are provided for entering External Descriptive Elements. The selections in each box are identical; W-wheels, S-straps, P-pockets, H-handle, C-combination lock, B-single item in a box and X for no external descriptive elements. These may be entered by pointing to the descriptive element in the drop-down list and clicking on the selected element or by placing the cursor in the field and entering the first letter of the descriptive element; W, S, P, H, C, B or X.

Manufacturer

 

Select the name of the luggage manufacturer by pointing to the name in the drop-down list and clicking on the selected manufacturer.
