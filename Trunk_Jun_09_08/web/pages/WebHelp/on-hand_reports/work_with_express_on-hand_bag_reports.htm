Work with Express On-hand Bags

 

 

An On-hand Baggage Report should be completed immediately for each Lost/Delayed bag so that Net-Tracer can begin the matching process. An Express On-hand Bag Report is a shortened version of the original form that only displays the required fields from each section. All other fields are set to their default value.

 

Regardless of which type of report is originally created, there is no difference between the two once the report is saved.

 

To add a new Express On-hand bag report:

  1. Log in to the Net-Tracer application.
     

  2. Select Express On-hand Bag from the On-hand menu.
     

  3. Enter the optional and required information in the appropriate fields for each section of the report.
     

  4. Scroll to the end of the report and click to save the report but not begin tracing the bag.

    - or -

    Click to save the report and begin to match the new record with all open On-hand bags.
     

Baggage Information Fields

Note

 

An asterisk (*) indicates the field is a required field.

 

Tip

 

Text box fields can be edited by typing over the existing data with new data. Dropdown list fields can be changed by moving your cursor over the new option to highlight it and clicking to enter the new data in the field.

 

Field Name

Field Description

Date/Time Created

The Date and Time is computer generated based on the date and time at the station where the report is created.

Station Found At

Station Found At is a locked field. The station code will default to the station assigned in your user profile.

Bag Tag Number

Enter the complete bag tag number. If the full bag tag number is not available enter any portion of the tag that is known. Net-Tracer will trace based on the string of characters available and provide potential matches. Leave the field blank if the bag tag number is unknown.

Color

 

Select the color of the bag by pointing to the color in the drop-down list and clicking on the selected color.  Alternatively, the color can be selected by placing the cursor in the field and entering the first letter of the desired color. For example, repeatedly entering the letter B will in order select Black, Blue, Beige and then Brown.

Type

Select the baggage type by pointing to the type in the drop-down list and clicking on the selected type. Alternatively, the type can be selected by placing the cursor in the field and entering the first digit of the desired bag type. For example, repeatedly entering the number 2 will in order select 20 through 29. For bag types 1-9 select 0 as the beginning number.
