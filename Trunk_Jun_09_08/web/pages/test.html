NetTracer Enhancement Release

December 2, 2010

Scan Data Update
Scan data will now show the actual tag number in the displayed results in order to relay accurate scan information. Please note, the leading digit will only be shown via Scan Data, not through any other source such as the Quick Search option.
 
Helpful Reminder:
 
The location of where the bag was checked is determined by the leading check Digit.
 
The following list shows where the bag was checked by this digit:
 
0 = Ticket Counter
1 & 6 =Gate Valet 
2 & 9 = Gate Checked 
3 = Crew Tag (U.K. use only)
4 = Manual Tag                                                                                            

 

Quick Search Date Range

Quick Search results will now automatically be limited to searching within the last 60 days, with a notice indicating the search date range.

CTRL + S brings up the Quick Search
